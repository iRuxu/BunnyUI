Pagination

< Source Code >
1.引入JS文件 
import Pagination from './module/Pagination.js'
2.创建实例 
var test = new Pagination({
    container : '.demo',    //加载容器
    total : 28,             //总页数
    limit : 10,             //每组最多显示页数
    start : 2               //起始页
})
3.实例方法 
test.onchange(callback(page,ele))
//当发生页码变更时，page为新的页码数，ele为点击的DOM元素，回调中this为实例对象，可通过._to(page)方法自定义页面跳转
4.引入CSS文件并自定义外观
